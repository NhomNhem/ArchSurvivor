# ArchSurvivor ⚔️
A 3D Hyper-casual RPG built with advanced Unity Architecture.

## 🚀 Tech Stack (Senior Standard)
This project demonstrates a scalable, testable, and decoupled architecture using modern Unity libraries:

* **Dependency Injection:** VContainer (Strict Lifetime Scopes).
* **Instantiation:** Sisus Init! (Constructor Injection for MonoBehaviours).
* **Reactive Programming:** R3 (Reactive Extensions for C#).
* **Async Handling:** UniTask (Zero allocation async/await).
* **Optimization:** ZString (Zero allocation strings).
* **UI:** UI Toolkit + DOTween (MVVM/MVP pattern).

## 🎯 Architecture Highlights
* **MVP Pattern:** Strict separation of logic (Model), data, and visuals (View).
* **Event-Driven:** Decoupled systems using Reactive Properties.
* **Zero-Alloc:** Heavily optimized for mobile performance.
